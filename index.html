Welcome to My API Project

This project represents my journey into the world of application architecture. Recently, during an interview for an architect position, I realized that I had never built a complete architecture from the ground up. This sparked the idea for creating my own API repository.

Project Description

My project involves creating an API repository that serves as a primitive library for Android platforms. The idea was to build something from scratch, to understand the intricacies of application architecture.

Why Slavic Mythology?

While there are numerous APIs based on Greek, Roman, and Norse mythologies, Slavic mythology remains largely untapped. I decided to focus on Slavic myths to bring to light the rich and diverse tales of Slavic culture. This choice also aligns with my personal interest and heritage, offering a unique perspective in the world of API development.

Looking Forward

The project is still in its infancy, with a basic test application developed for initial trials. Future plans include expanding the API's capabilities and exploring more aspects of Slavic folklore. I invite collaboration and feedback to grow this project further.
